{{.App.Name}} · 提交本机 UDID

把你 iPhone 的 UDID 发给开发者，加进 Provisioning Profile 后就能正常安装应用。

方法 1：iOS 18 / iOS 17 设置里查（最快）

  1. 打开 设置 → 通用 → 关于本机
  2. 找到 序列号 那一行，长按它
  3. 弹出菜单里选 "复制 UDID"（不是"复制序列号"）
  4. 回到本页面，粘贴到下面输入框

方法 2：用 Mac Finder（旧版 iOS）

  1. iPhone 用线连 Mac
  2. 打开 Finder，左栏选你的 iPhone
  3. 顶部信息条点击 "型号 / 容量 / 序列号" 那一行会循环切换，切到 UDID
  4. 右键 → 拷贝
