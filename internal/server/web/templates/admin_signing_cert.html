文件 + 密码不会写到磁盘明文。distsrv 解开 .p12 后只把 cert 链和私钥（PEM 格式）保存到 {{.Site.OrgName}} 服务器的数据目录（权限 0600，只 distsrv 用户能读）。

如何在 Apple Developer 导出 .p12

  1. Mac 打开 钥匙串访问（Keychain Access）
  2. 左栏选 登录 → "我的证书"
  3. 找到 Apple Distribution 或 Apple Development 那一行（前面三角展开，里面有对应的私钥）
  4. 同时选中证书 + 私钥（按 Cmd 多选）
  5. 右键 → "导出 2 个项目" → 文件格式选 Personal Information Exchange (.p12)
  6. 设个密码，保存为 profile-signing.p12
  7. 把这个 .p12 上传到本页面

同一份 .p12 同样可用于签 ipa；distsrv 这里只读它的 cert + 私钥来 PKCS7 签 mobileconfig。
